# Simon Says Game

A simple Simon Says memory game built using HTML, CSS, and JavaScript.

## About

This project recreates the classic Simon Says game where players must remember and repeat an increasing sequence of button flashes. The game becomes more challenging as the sequence grows longer.

## Features

- Interactive gameplay
- Random sequence generation
- Score tracking
- Highest score tracking
- Button flash animations
- Keyboard-controlled game start

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Play

1. Press any key to start the game.
2. Watch the button sequence carefully.
3. Repeat the sequence by clicking the buttons in the correct order.
4. Each level adds a new step to the sequence.
5. The game ends when an incorrect button is clicked.

## Learning Objectives

This project was created to practice:

- DOM Manipulation
- Event Handling
- JavaScript Arrays
- Game Logic Implementation
- CSS Animations and Styling
